# BloomCart

Android online shopping application built with **Kotlin** and **Jetpack Compose**.  
Features Firebase Authentication, Hilt dependency injection, Room local database, product categories, discount badges, detailed product views, and powerful search.

## Features
- Modern UI with Jetpack Compose
- Homepage showing product categories & discount badges
- Product listing with search functionality
- Detailed product screen
- Secure login/signup using Firebase Authentication
- Offline support & local storage with Room database
- Dependency injection via Hilt

## Tech Stack
- **Language**: Kotlin
- **UI**: Jetpack Compose
- **Dependency Injection**: Hilt
- **Authentication**: Firebase Auth
- **Local Database**: Room

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Nusrat1604063/BloomCart-E-Commerce-App.git

2. Open the project in Android Studio.
3. Add your Firebase google-services.json file to the app/ folder (get it from Firebase Console).
4. Sync Gradle (File → Sync Project with Gradle Files).
5. Build and run the app on an emulator or device.
